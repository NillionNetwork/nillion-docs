# API Access

The nilPay Portal allows you to register directly on the nilDB nodes and getting ready to use the [Private Storage](/blind-computer/build/storage/overview) network APIs and SDK. This guide walks you through the process.


## Prerequisites
You must have an EVM wallet address and browser extension (MetaMask, Rabby, etc)

## Getting Started

Open up the [nilPay Portal](https://subscription.nillion.com) and sign in via your desired EVM browser wallet.

![nilPay Page](/img/nilpay-sign-in.png)

## Registering on nilDB nodes
You can now register directly on the nodes of your preference. We recommend at least 3 nodes for your Private Storage cluster.

![nilPay Page](/img/nilpay-mainnet.png)

## Free Tier & Topping up
You have 100MB of free storage on each node. Beyond the free tier, you should top up your credits on each node using NIL - the rate per GB/hr is displayed on the portal.
- To obtain NIL you can buy or swap to NIL on a decentralised exchange (for example Uniswap) or directly from your wallet browser extension
- You will also need a tiny amount of ETH for gas fees
![nilPay Page](/img/nilpay-topup.png)
