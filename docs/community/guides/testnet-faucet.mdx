# Testnet Faucet Guide

This guide walks you through getting testnet NIL from the [Nillion Faucet](https://faucet.testnet.nillion.network/). You will need an EVM wallet (e.g. MetaMask or Rabby) with Sepolia ETH to cover gas, then you can claim testnet NIL from the faucet.

## Step 1: Create an EVM wallet

Create an EVM-compatible wallet if you don't have one. Common options include:

- [MetaMask](https://metamask.io/)
- [Rabby](https://rabby.io/)

Install the browser extension (or mobile app), create or import a wallet, and make sure you have your recovery phrase stored securely.

## Step 2: Add the Sepolia network

You will need Sepolia (Ethereum testnet) in your wallet to receive testnet ETH and to use the Nillion faucet.

### Manual setup

1. Open your wallet and go to **Settings** or the network dropdown.
2. Click **Add network** or **Add a network manually**.
3. Enter the following details:

<div style={{marginLeft: '1.5rem'}}>

- **Network Name:** Sepolia
- **New RPC URL:** https://ethereum-sepolia-rpc.publicnode.com
- **Chain ID:** 11155111
- **Currency Symbol:** ETH
- **Block Explorer:** https://sepolia.etherscan.io

</div>

4. Save the network and switch to Sepolia.

## Step 3: Get Sepolia ETH

You need a small amount of Sepolia ETH to pay for gas when claiming NIL from the faucet.

1. Visit a Sepolia faucet, such as the [Conduit Faucet](https://faucet.conduit.xyz/) or [Alchemy Faucet](https://www.alchemy.com/faucets/ethereum-sepolia).
2. Connect your wallet and request Sepolia ETH.
3. Wait for the transaction to complete.

For more detail, see [Get Sepolia ETH](/blacklight/tools/faucets#step-1-get-sepolia-eth) in the Blacklight faucets guide.
