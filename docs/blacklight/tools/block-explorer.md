# Block Explorer

Block explorers allow you to view transactions, blocks, addresses, and other onchain data on Nillion's Ethereum L2. Use the block explorer to track your transactions, verify contract deployments, and explore the network.

## Mainnet

The block explorer for mainnet can be viewed at: [https://explorer.nillion.network](https://explorer.nillion.network).

## Testnet

The block explorer for mainnet can be viewed at: [https://explorer.testnet.nillion.network](https://explorer.testnet.nillion.network).
