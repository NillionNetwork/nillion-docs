# Set Up a Wallet

Blacklight is an EVM-compatible network, which means you can connect to it using any Ethereum-compatible wallet. This guide will help you add Blacklight to your wallet.

## Adding Blacklight to Your Wallet

1. Open your wallet extension or app
2. Navigate to network settings (usually found in settings or network dropdown)
3. Click "Add Network" or "Add Custom Network"
4. Enter the network details from the tables below
5. Save the network
