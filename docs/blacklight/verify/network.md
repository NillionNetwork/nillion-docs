# Network Configuration

Users running Blacklight verifier nodes and developers registering their apps for verification via Nillion Blacklight must connect to Nillion's Ethereum L2. The network works like any other EVM chain.

## Mainnet

### Network

| Property        | Value                                           |
| --------------- | ----------------------------------------------- |
| Chain Name      | Nillion Network                                |
| RPC URL         | https://rpc.nillion.network                     |
| Web Socket URL  | wss://rpc.nillion.network                       |
| Chain ID        | 98875                                           |
| Currency Symbol | ETH                                             |
| Block Explorer  | https://explorer.nillion.network               |
